Go to Actions tab > Run workflow — it generates today's issue into /issues/YYYY-MM-DD/ 4. Netlify auto-deploys, your temp link like drumbeat-test-2026.netlify.app shows fresh news. **Cost for trial:** - RSS + FRED + CoinGecko + ESPN = $0 - GitHub Actions = free 2000 min/month - Netlify second site = free - Total for 2-3 day trial = $0 **Netlify hosting without affecting live site:** Netlify allows multiple sites.
